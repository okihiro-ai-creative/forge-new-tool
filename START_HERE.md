# START_HERE: Forge New Tool

このリポジトリをClaude Code等のAIコーディングエージェントに渡し、SKILL.mdとSTART_HERE.mdを読んで実行してもらってください。

## 前提

- Python 3.10以上を用意する。
- このフォルダを任意のプロジェクトに配置する。
- 外部APIは標準では不要。生成するツールがAPIを使う場合は、利用者自身のAPIキー・アカウント・認証情報を用意する（BYOK）。
- ローカルGPU等のハードウェア要件は標準では不要。生成するツール側で必要になる場合は、そのツールの要件として確認する。

## セットアップ

1. AIエージェントに `SKILL.md` を読ませる。
2. 作りたいツールの目的、入力、出力、実行環境を伝える。
3. 生成先はプロジェクトルート直下の `tools/` または `apps/` を推奨する。

## 動作確認

AIエージェントに小さなCLIツールを1つ作らせ、`python build.py`、セットアップ、起動確認まで実行させる。

このツールのカスタマイズや、似たツールの新規開発を相談したい場合は、README.md末尾のリンクを確認してください。
